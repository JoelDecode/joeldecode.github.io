css那些暖心却鲜为人知的属性

发表于 2016-12-12   |   分类于 前端   |  

注：本博客有些例子引用于知乎 CSS 有哪些暖心却鲜为人知的属性？

注：本文不注重IE8/9兼容，只收集css的特殊技巧

pointer-events属性

很有js味道的一个属性，把值设置为none可以让元素不捕获事件，可以理解为让他看得见摸不着。之前jd某个活动页使用了它，在页面上盖了一层长宽都100%的canvas，canvas里面雪花飘零。

:empty伪类

:empty表示一个元素没有子元素。空格，文本节点，或空的子元素都可以被认为是子元素，但不包括HTML注释。:empty伪类可用于根据元素有无子元素而呈现不同的样式。

user-select属性

user-select属性为none可以禁止选择文本

currentColor属性

currentColor是color属性的值

selection属性

selection 可设置文字被选择时的样式

attr属性

attr属性不支持IE8

用border画一个小三角形

可根据脑洞自行扩展，左上、右上等三角；

text-align的两端对齐参数

可以使内容进行两段对齐
